String s = "1";
while (s != "1000") {
s += "0";
}

Esto es porque en porque en Java el operador == compara que ambos tengan la misma 
referencia, no el mismo contenido, por lo que |s != "1000"| siempre sera True ya que
este nuevo string "1000" nunca tendra la misma referencia que el String s, ocacionando asi
el bucle infinito.

Mientars que en C# el == si que puede comparar el contenido de ambos objetos por ende no 
seria un bucle infinito.
